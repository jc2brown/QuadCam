Microblaze C code project


To begin contributing, you must open the Xilinx SDK, then set your workspace and import the existing project.

1. File > Switch Workspace > Other...
	Set the workspace folder to be the git root folder, ie. 'QuadCam' and not 'QuadCamSW'
2. Import the existing project into your workspace. You do NOT need to select 'Copy projects into workspace'.
	http://www.xilinx.com/support/documentation/sw_manuals/xilinx12_2/SDK_Doc/tasks/sdk_t_import_exist_proj.htm
3. Close the welcome message tab if it is still there. The project files should appaer in the Project Explorer window.
	( The main() function can be found in quadcam.c )
